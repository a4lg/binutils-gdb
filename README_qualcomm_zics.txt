Summary
========

This is the Qualcomm's Code Size Reduction Instructions implementation
with tentative extension name 'Zics'.


Missing
========

PC-relative load/store instructions are not implemented since it will
require the new type of relocation.


Commits
========

This is fine-grained than my usual patch sets for faster prototyping.
